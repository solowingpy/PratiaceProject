#运算符小总结

**这个文件主要记录一些运算符和三元运算符的细节**

1. 复合运算符进行运算时会自动进行数据类型的转换，比如下面的写法：

   > int num = 1; num += 3.5
   >

   正常情况下，num和3.5进行运算后整个结果的精度提升到了double类型，如果在用整型的num承接结果的话，就会报错

   但是如果用复合运算符(像“++”，“+=”，“/=”)进行运算，就会把运算结果强制转换成正确的数据类型。
